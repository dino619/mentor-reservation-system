# Mentor Reservation System Frontend

This folder contains the Vue 3 frontend for the Mentor Reservation System prototype.

The full project documentation is in the root `README.md`.

## What It Provides

- prototype login by selecting a seeded student or mentor,
- student registration and simple login,
- student dashboard for browsing mentors and sending requests,
- mentor dashboard for accepting or rejecting requests,
- request status tracking,
- in-app notifications,
- API integration with the ASP.NET Core backend.

## Technologies

- Vue 3
- Vite
- Vue Router
- Fetch API

## Run Frontend

Start the backend first on `http://localhost:5266`, then run:

```powershell
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## Build

```powershell
npm run build
```
